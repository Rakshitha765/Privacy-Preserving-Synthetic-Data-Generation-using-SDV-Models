# Dataset

This project uses the **Adult Income Dataset** downloaded from **Kaggle**.

**Dataset Information**
- Source: Kaggle
- Records: 5,000
- Features: 15
- Type: Tabular Dataset

The dataset contains a combination of numerical and categorical attributes and is used to evaluate privacy-preserving synthetic data generation using SDV models.

> Note: The dataset is not included in this repository. Please download it from Kaggle if you want to reproduce the project.
